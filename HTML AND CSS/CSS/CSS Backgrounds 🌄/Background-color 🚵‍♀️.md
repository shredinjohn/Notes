## Background-color 

```css
h1 {  background-color: green;}  
  
div {  background-color: lightblue;}  
  
p {  background-color: yellow;}
```

## Opacity/Transparency 

```css
div {
background-color : green;
opacity : 0.3 ; /* the lower the less visible */
}
```

Also,

```css
div {
background-color : rgba(255,255,0,0.3);
opacity : 0.3 ; /* the lower the less visible */
}
```
