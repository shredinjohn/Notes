## Colspan and Rowspan Examples

```html
<table border="1px solid black">
  <thead>
    <tr>
      <td>Quick</td>
      <td colspan=2>brown fox</td>
      <td> jumps</td>
    </tr>
  </thead>
  <tr>
    <td rowspan=3>over the</td>
    <td>lazy</td>
    <td>dog</td>
    <td>and</td>
  </tr>
  <tr>
    <td>then</td>
    <td>it</td>
    <td>fall</td>
  </tr>
  <tr>
    <td colspan=3>
      prey to a lion
    </td>
  </tr>
</table>

```
